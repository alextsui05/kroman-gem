
# Kroman

Kroman is ultimate Korean hangul romanization tool. It has a lot of
implementations.

This is the ruby gem.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'kroman-gem'
```

And then execute:

$ bundle

Or install it yourself as:

$ gem install kroman-gem

## Usage

## Usage

``` ruby
Kroman.parse(희망은
=> heui-mang-eun

Kroman.let_me_sing!(손목시계)
=> son-mog-si-gye
```
